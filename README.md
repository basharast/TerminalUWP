# Features

- Re-Design the layout
- Commands History
- Scroll Mode
- Encrypted Profiles
- Save output as text
- Bugs fixes (including font issue)


# Target

It will work with build 16299 and above, ARM - x64 - x86

For Windows Phone users (if you still have one) it will work on build 1709 (15254) only
